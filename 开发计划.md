4. 填寫項目信息：

6. 等待2-3分鐘讓Supabase創建項目

## 🔐 階段三：Google 登入設置（需要你配合）
**進度：0/4 步驟完成**

### 步驟 3.1：創建 Google OAuth 應用
**你需要做**：
1. 打開瀏覽器，訪問 https://console.cloud.google.com
2. 登入你的Google帳號
3. 點擊頂部「Select a project」→「New Project」
4. 項目名稱填：`ai-day-auth`（直接複製）
5. 點擊「Create」
6. 等待幾秒鐘

**我會告訴你**：每一步的具體操作

### 步驟 3.2：啟用 Google+ API
**你需要做**：
1. 在Google Cloud Console左側菜單，找到「APIs & Services」→「Library」
2. 搜索「Google+ API」
3. 點擊進入，點擊「Enable」按鈕
4. 完成！

**我會告訴你**：具體路徑

### 步驟 3.3：創建 OAuth 憑證
**你需要做**：
1. 在左側菜單找到「APIs & Services」→「Credentials」
2. 點擊「Create Credentials」→「OAuth client ID」
3. 如果提示配置OAuth同意屏幕，點擊「Configure consent screen」
   - User Type選擇「External」
   - 點擊「Create」
   - App name填：`AI-Day`（直接複製）
   - User support email選你的郵箱
   - Developer contact填你的郵箱
   - 點擊「Save and Continue」3次，最後點「Back to Dashboard」
4. 回到Credentials頁面，再次點擊「Create Credentials」→「OAuth client ID」
5. Application type選擇「Web application」
6. Name填：`AI-Day Web`（直接複製）
7. Authorized redirect URIs：我會告訴你填什麼（先留空，我會給你）
8. 點擊「Create」
9. 會彈出一個窗口，複製「Client ID」和「Client Secret」給我

**我會告訴你**：每一步的具體操作和要填的內容

### 步驟 3.4：在 Supabase 中配置 Google 登入
**你需要做**：
1. 回到Supabase項目
2. 左側菜單找到「Authentication」→「Providers」
3. 找到「Google」，點擊開啟
4. 把我給你的Client ID和Client Secret填進去
5. 點擊「Save」

**我會告訴你**：具體每一步

**階段三完成後**：用戶就可以用Google帳號登入了！
